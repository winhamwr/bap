 = Installation =

On a fresh ubuntu 8.04 box. The variable MYPROJ will probably be something like bap

 1. Download this file: http://svn.pinaxproject.com/pinax/trunk/scripts/pinax-boot.py and stick it in your workspace
 2. run:
{{{
python pinax-boot.py --development ~/.virtualenvs/MYPROJ
}}}
 3. run:
{{{
source MYPROJ-env/bin/activate
}}}
 4. Install PIL:
   * Download and extract: http://effbot.org/downloads/Imaging-1.1.6.tar.gz
   * Browse to the extracted folder and run
{{{
python setup.py install
}}}
